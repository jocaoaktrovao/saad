ALGORITMOS DE ORDENAÇÃO

4. MERGE SORT

O Merge Sort é um algoritmo de ordenação recursivo que divide a lista ao meio, ordena cada metade separadamente e, em seguida, combina as duas metades ordenadas. Esse processo de divisão e combinação é repetido até que a lista completa esteja ordenada. O Merge Sort é muito eficiente e tem um tempo de execução O(n log n), tornando-o uma boa escolha para listas grandes.
