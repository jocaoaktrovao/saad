ALGORITMOS DE ORDENAÇÃO

3. INSERTION SORT

O Insertion Sort constrói a lista ordenada elemento por elemento, sempre inserindo o próximo elemento na posição correta. Ele percorre a lista e, para cada elemento, move-o para a posição correta em relação aos elementos anteriores. É eficiente para listas pequenas ou quase ordenadas, mas, assim como o Bubble e Selection Sort, não é ideal para listas grandes.
