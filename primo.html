VERIFICAR SE UM NÚMERO É PRIMO COM ESCOLHA DO USUÁRIO

CÓDIGO:


Saída do código aparecerá aqui...

Explicação

Objetivo: Verificar se um número n é primo.

Processo:
Se n for menor ou igual a 1, não é primo.
O loop percorre todos os números de 2 até a raiz quadrada de n. Se n for divisível por qualquer desses números, ele não é primo.

Entrada do Usuário:
O usuário insere um número para verificar se é primo.

Verificação e Exibição do Resultado:
O programa verifica se o número é primo e exibe o resultado.
